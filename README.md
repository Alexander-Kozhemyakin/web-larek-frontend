# Проектная работа "Веб-ларек"

Стек: HTML, SCSS, TS, Webpack

Структура проекта:
- src/ — исходные файлы проекта
- src/components/ — папка с JS компонентами
- src/components/base/ — папка с базовым кодом

Важные файлы:
- src/pages/index.html — HTML-файл главной страницы
- src/types/index.ts — файл с типами
- src/index.ts — точка входа приложения
- src/scss/styles.scss — корневой файл стилей
- src/utils/constants.ts — файл с константами
- src/utils/utils.ts — файл с утилитами

## Установка и запуск
Для установки и запуска проекта необходимо выполнить команды

```
npm install
npm run start
```

или

```
yarn
yarn start
```
## Сборка

```
npm run build
```

или

```
yarn build
```

## Описание базовых классов

Базовый интерфейс для всех компонентов.

Обеспечивает стандартную структуру для всех компонентов с методами рендера и привязки событий.

Методы:
- render(): HTMLElement — рендеринг компонента.
- bindEvents(): void — привязка событий к элементам компонента.

## Описание компонентов

### Компонент Header
Использует интерфейс IHeader для типизации данных cartCount.
- Отображает и обновляет счётчик товаров в корзине.

### Компонент ProductCard
Связан с модальным окном через IProductModal
Функции:
- Отображает карточку товара (заголовок, категория, картинка, цена).
- Реализует кликабельность для отображения подробной информации о товаре в модальном окне.

### Компонент Cart
Связан с компонентом кнопки оформления заказа.
Функции:
- Отображает список товаров, добавленных в корзину.
- Показывает общую стоимость.
- Обрабатывает удаление товаров из корзины.

### Компонент Modal
Использует интерфейс IModalComponent для стандартного рендера и обработки событий.
Функции:
- Отображает контент в модальном окне.
- Обеспечивает закрытие окна через кнопку или клик вне окна.
